# nba98爬虫

#### 介绍
将nba98的信息爬取并保存为json和excel文件,可以爬取大到省市区,小到街道居民楼的小姐姐信息,也许她就在你的身边,也许她就是你的女神~

#### 软件架构
python


#### 安装教程

```bash
git clone https://gitee.com/first-love-cloud/nba98.git && cd nba98 && pip install -r requirements.txt
```

#### 使用说明

##### 1. 获取省市区
```bash
python city.py
```
##### 2. 获取小姐姐信息
```bash
python main.py
```
